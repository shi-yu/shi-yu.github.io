hexo博客搭建

第一步：安装nodeJS

1.打开浏览器进入https://nodejs.org/en/

2.下载nodejs并安装

3.在终端输入node -v测试是否安装成功

第二步：安装hexo

1.在终端状态下输入 sudo npm install hero -g

2.输入密码*

3.输入hexo回车进行检测

4.创建目录mkdir blog,并进入 cd blog目录

5.输入 hexo init 初始化hexo目录

6.输入 hexo server 启动本地服务

第三步：配置github

1.在终端blog目录下配置ssh-key：输入 ssh-keygen,连续三次回车

2.再输入vim ~/.ssh/id_rsa.pub ,复制ssh-key

3.登陆github.com,注册账号和用户名，用户名是域名前缀，不要乱设，设好后登陆

4.进入后右上方有个绿色的方块 Edit profile，点击进入

5.选中左边竖目录栏SSH and GPS keys，再点击右边New SSH key，Tiele里面输入一个标题，把ssh-key复制到key里面，点击Add SSH key

6.在终端里输入git config –global user.name”shi-yu”

7.在终端里输入git config –global user.email”*“

8.新建博客域名仓库 点击右上方的加号，第一个选项，name一定要正确的输入，用户名.github.io，下方第一个和第三个复选框勾上，确定提交，然后点击右上方的Clone or download，复制地址

第四步：配置blog文件

1.用sublime text打开config.yml :在最后找到deploy,type,repe
type空格加上git，repo空格加上域名仓库的https地址，保存！

第五步：上传文件

1.在终端bolg目录下输入npm install hexo-deployer-git–save保存本地做好的文档

2.继续输入hexo g打包成html文件

3.最后输入hexo d 生成！

自适应第一节.md blog
