css3选择器

子元素选择器 vs 后代选择器

E>F VS E F

  • 子元素选择器 选中的是子元素F 子孙不会被选

    .div1>div{
        border:1px solid red;
        color:red;
}

  • 后代选择器 是选择所有的F 包括子子孙孙

    .div1 div{
        border:1px solid red;
        color:red;
}

相邻兄弟选择器 VS 通用兄弟选择器

E + F VS E ~ F
  • 都要统一父级，是同级的
  • 相邻兄弟选择器的重点是在相邻，通用选择器不需要
  • 在上方的不能适用

  • 相邻兄弟选择器有且只有一个

    h2+div{
    border:1px solid green;
}

  • 通用兄弟选择器选择多个符合的

h2~div{
    border:1px solid green;
}

属性选择器

E[attr]
E[attr="value"]
E[attr~="value"] ~:表示几个以空格隔开的属性值中包含了“value”
E[attr^="value"] ^:以value开头的值
E[attr$="value"] $：以value结尾的值
E[attr*="value"] *： 包括了value的值
E[attr|="value"] |：以value或value-开头
E[attr="value"]和E[attr*="value"]是最实用的

*[class]{
    border:1px solid pink;
}
*[class="div1"]{
    border:1px solid pink;
}
*[class~="div1"]{
    border:1px solid pink;
}
*[class^="div1"]{
    border:1px solid pink;
}
*[class$="div1"]{
    border:1px solid pink;
}
*[class*="div1"]{
    border:1px solid pink;
}
*[class|="div1"]{
    border:1px solid pink;
}

：nth 选择器

  • ：fist-child选择某个元素的第一个子元素
  • ：last-child选择某个元素的最后一个子元素
  • ：nth-last-child()选择某个元素的一个或多个特定的子元素，从这个元素的最后一个子元素开始算；
  • ：nth-of-type()选择指定的元素；
  • ：nth-last-of-type()选择指定的元素，从元素的最后一个开始计算；
  • ：first-of-type选择一个上级元素下地第一个同类子元素；
  • ：last-of-type选择一个上级元素的最后一个同类子元素；
  • ：only-child选择的元素是它父元素的唯一一个子元素；
  • ：only-of-type选择一个元素是它的上级元素的唯一一个相同类型的元素；
  • ：empty选择的元素里面没有任何内容。
  • ：nth-child()选择某个元素的一个或多个特定的子元素；
  • ：nth-child(length);参数是具体数字；
  • ：nth-child(n);参数是n,n从1开始计算
  • ：nth-child(2n);n的倍数选择，n从1开始算；
  • ：nth-child(n+length);选择大于length后面的元素；
  • ：nth-child(-n+length);选择小于length前面的元素；
  • ：nth-child(2n+1);表示隔几选一

否定选择器（：not）

ex:input:not([type="submit"]){border:1px solid red;}

伪元素

  • ：：first-letter 选择文本块的第一个字母，除非在同一行里面包含一些其他元素，不过这个主要运用于段落排版上
  • ：：first-line 选择元素的第一行
  • ：：before，：：after 这两个主要用来给元素的前面或后面插入内容，这两个常用“content”配合使用，见过最多的就是清除浮动还有图片icon

  • ：：selection用来改变浏览网页选中文本的默认效果，“：：selection”来改变在浏览器中选中文本后的背景色与前景色。：：selection在IE家族中，只有IE9+版本支持，在Firefox中需要加上其前缀“-moz”, :”::selection”只能设置两个属性，一个就是background,另一个就是color属性，设置其他属性是没有任何效果的。

    /*伪元素*/
body::before{
    height:20vh;
    content:'12';
    display: block;
    background-color:pink;
}
/*只能写文本内容,子元素的最上方*/
body::after{
    content: '个人看过节日快乐两个';
    background-color: red;
    display: block;
    height:20vh;
}
/*子元素的最下方*/
.div1{
    height:60vh;
    background-color: yellow;
}
.div1::first-letter{
    background-color:red;
}
.div1::first-line{
    color: green;
}
.div1::selection{
    background-color:red;
color:yellow;
}
/*改变选中文本的背景和颜色*/
